\doxysubsection{Detailed Description}
There is one instance of this structure for each of the core S\+P\+Is. Instances of this structure are used to identify a specific S\+PI. A pointer to an instance of the spi\+\_\+instance\+\_\+t structure is passed as the first parameter to S\+PI driver functions to identify which S\+PI should perform the requested operation.
