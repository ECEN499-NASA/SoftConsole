◆ get_dec_from_user()

uint32_t get_dec_from_user ( uint8_t  numDecPlaces)

Gets an integer value from the user.

Function will only return a correct value if it is smaller than the max value represented with 32bits. If the user enters more than the number of placed specified, the function will ignore them.

printf("What is 2 + 2?\n");
uint32_t input = get_dec_from_user(2);
// If the user's input is "004", then the returned value would be 0
// If the user enters "4" or "04", then the returned value would be 4
if(input != 4)
printf("Go back to kindergarden\n");
else
printf("Good job! Here is a cookie\n");
Parameters
numDecPlacesNumber of decimal places to be received by the user
Returns
User input of uint32_t type

◆ get_string_from_user()

void get_string_from_user ( uint8_t  numChars,
uint8_t  spaceEnabled,
char *  input 
)

Gets a string of characters from the user.

Doesn't register the user pressing the ESC_KEY, BACKSPACE_KEY, or TAB_KEY. When the user has entered the maximum amount of characters, or has pressed the ENTER_KEY, it will end the function. If param "spaceEnabled" is false, it will ignore that to

#define MAX_INPUT 10
uint8_t numChars = MAX_INPUT;
char input[MAX_INPUT];
uint8_t spaceEnabled = 0x01;
get_string_from_user(numChars, spaceEnabled, &input);
Parameters
numCharsThe maximum number of characters the user can enter
spaceEnabled1 : Accepts spacebar input 0 : Ignores spacebar input
inputPointer/array to be filled with user input. Array's \ size is defined by the "numChars" param
